# Crustchan2 Frontend-ng

This is the frontend of crustchan2 utilizing the [tuono](https://tuono.dev/) project.  This project should take priority over getting the API to functional completion.  API Features can be pulled out and if needed, we can facilitate communication between the backend of tuono and the API rather than direct DB access at a later point.


IMPORTANT NOTES:
* Because of the Tuono version in use and the use of dynamic routes, it is "Only available for SSR apps. This feature will be soon available also for static generated apps."  This should get upgraded when support arrives.
